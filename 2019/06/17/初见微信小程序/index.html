最近几年微信小程序的开发逐渐成为一种新趋势，之前做了一个简单的小程序，把开发微信小程序
过程中的一些教训和经验记录下来。

小程序解决方案的安装

小程序注册

打开微信公众平台注册并登录小程序，按如下步骤操作：

  1. 单击左侧菜单栏中的【设置】。
  2. 单击右侧 Tab 栏中的【开发者工具】。
  3. 单击【腾讯云】，进入腾讯云工具页面，单击【开通】。
  4. 使用小程序绑定的微信扫码即可将小程序授权给腾讯云，开通之后会自动进去腾讯云微信小程
    序控制台，显示开发环境已开通，此时可以进行后续操作。

安装开发工具

下载并安装最新版本的微信开发者工具
，使用小程序绑定的微信号扫码登录开发者工具。

创建demo

打开第二步安装的微信开发者工具，按如下步骤操作：

  1. 选择【小程序项目】按钮。输入小程序 AppID，项目目录选择一个空的目录，接着选择【建立
    腾讯云 Node.js 启动模板】，点击确定创建小程序项目。

  2. 安装依赖。在刚刚选择的目录打开 CMD 安装依赖。

  3. 点击界面右上角的【腾讯云】图标，在下拉的菜单栏中选择【上传测试代码】。

  4. 选择【模块上传】并勾选全部选项，然后勾选【部署后自动安装依赖】，点击【确定】开始上传代码。
  5. 上传代码完成之后，点击右上角的【详情】按钮，接着选择【腾讯云状态】即可看到腾讯云
    自动分配给你的开发环境域名。
  6. 完整复制（包括 https://）开发环境 request 域名，然后在编辑器中打开 client/config.js
    文件，将复制的域名填入 host 中并保存，保存之后编辑器会自动编译小程序，左边的模拟器窗口
    即可实时显示出客户端的 Demo
  7. 在模拟器中点击【登录】，看到显示“登录成功”，即为开通完成。

小程序开发简析

前端开发

前端开发文件结构如下图所示：

  1. 在config.js中设置Server端通信地址以及通信配置，以app.js作为小程序入口，在app.json中
    注册各个前端界面的路由。
  2. 在pages下建立并编写实现各个前端界面的View层和App Server层。
  3. utils文件夹下放置要经常用的工具类API，供各个界面的App Server层，demo中已包含一个常用
    拟态框API（util.js）。
  4. 其中Color-UI组件库和EC-Canvas图表插件只是我在项目中用到了，大家可以不用或选择更合适自己的。

后端开发

wafer2快速开发解决方案后端配备的是koa2框架，下面主要介绍node.js版本结构：

  1. 在config.js中设置端口、微信小程序的App ID以及App Secret，设置数据库连接配置，并设置
    微信登录态有效期。
  2. 以app.js作为Server端入口，引入路由分发routes和响应处理中间件middlewares，启动程序
    并监听端口。
  3. 在routes下处理前端发送过来的网络请求，并分发到controllers下对应的API接口中进行处理。
  4. controllers下编写实现已设计好的API接口，API接口处理结果和要返回的数据经过响应处理中
    间件middlewares下的response.js进行错误捕获和处理后返回给Client端。
  5. controller中涉及到对数据库的操作，可以通过Knex.js数据库查询构造器进行处理（可选）。
  6. 在tools下使用数据库初始化脚本initdb.js对数据库进行初始化，初始化完成后，读取cAuth.sql
    文件自动创建会话管理用户信息表。
